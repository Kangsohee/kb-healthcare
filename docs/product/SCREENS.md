# Screen Layouts

각 페이지의 레이아웃 와이어프레임. 실제 픽셀 치수는 `docs/design/DESIGN.md` 참고.

---

## 공통 레이아웃 (인증 후 페이지)

```
┌────────────────────────────────────────────────────────────────┐
│ GNB (height: 56px, sticky top-0, shadow-sm)                    │
│  [logo / 서비스명]                             [UserCircle]     │
├──────────────┬─────────────────────────────────────────────────┤
│ LNB          │                                                 │
│ (width:240px)│  Content Area                                   │
│              │  padding: 24px                                  │
│ [LayoutDash] │                                                 │
│  대시보드     │                                                 │
│              │                                                 │
│ [CheckSquare]│                                                 │
│  할 일        │                                                 │
│              │                                                 │
│  ← 활성 메뉴: 좌측 border-primary + text-primary              │
└──────────────┴─────────────────────────────────────────────────┘
```

**반응형:**
- `< 768px`: LNB 숨김, 하단 네비게이션 또는 햄버거 메뉴
- `768px ~`: LNB icon only (48px)
- `≥ 1024px`: LNB 전체 (240px)

## /sign-in — 로그인

LNB 없음. GNB에 `[LogIn]` 아이콘.

```
┌────────────────────────────────────────────────────────────────┐
│ GNB                                              [LogIn]        │
├────────────────────────────────────────────────────────────────┤
│                                                                │
│              ┌──────────────────────────────────┐             │
│              │  로그인                           │             │
│              │                                  │             │
│              │  이메일                           │             │
│              │  ┌────────────────────────────┐  │             │
│              │  │ user@example.com           │  │             │
│              │  └────────────────────────────┘  │             │
│              │                                  │             │
│              │  비밀번호                         │             │
│              │  ┌──────────────────────────[Eye]│             │
│              │  │ ••••••••                   ]  │             │
│              │  └────────────────────────────┘  │             │
│              │                                  │             │
│              │  ┌────────────────────────────┐  │             │
│              │  │         로그인              │  │             │
│              │  └────────────────────────────┘  │             │
│              │  (유효하지 않으면 disabled 상태)  │             │
│              └──────────────────────────────────┘             │
│                                                                │
└────────────────────────────────────────────────────────────────┘
```

**에러 상태:**
```
│  이메일                                                        │
│  ┌────────────────────────────┐  ← border-error              │
│  │ invalid-email              │                               │
│  └────────────────────────────┘                               │
│  올바른 이메일 형식을 입력해주세요   ← text-error text-xs       │
```

**API 실패 모달:**
```
┌─────────────────────────────────────────┐
│  오류                              [X]  │
│  ─────────────────────────────────────  │
│  이메일 또는 비밀번호가 올바르지 않습니다. │
│                                         │
│                           [확인]        │
└─────────────────────────────────────────┘
```

## /task — 할 일 목록

```
┌──────────────┬─────────────────────────────────────────────────┐
│ LNB          │  할 일                                           │
│  [Task*]     │  (text-2xl font-bold)                           │
│              │                                                 │
│              │  ┌─────────────────────────────────────────┐   │
│              │  │  운동하기                          TODO  │   │
│              │  │  30분 이상 유산소                        │   │
│              │  └─────────────────────────────────────────┘   │
│              │  ← TaskCard (height: 96px, shadow-sm)           │
│              │                                                 │
│              │  ┌─────────────────────────────────────────┐   │
│              │  │  물 마시기                         DONE  │   │
│              │  │  2L 이상                                 │   │
│              │  └─────────────────────────────────────────┘   │
│              │  ← DONE: text-success                           │
│              │                                                 │
│              │  ... (가상 스크롤, viewport 내 아이템만 렌더링) │
│              │                                                 │
│              │  ┌── 다음 항목 불러오는 중 spinner ──┐           │
│              │  (hasNext=true일 때 하단 sentinel)              │
└──────────────┴─────────────────────────────────────────────────┘
```

**빈 목록:**
```
│              │                                                 │
│              │          등록된 할 일이 없습니다.               │
│              │                                                 │
```

## /task/:id — 할 일 상세

```
┌──────────────┬─────────────────────────────────────────────────┐
│ LNB          │  ← 목록으로                     [Trash2 삭제]   │
│              │  (ArrowLeft icon + "목록으로")   (text-error)    │
│              │  ─────────────────────────────────────────────  │
│              │                                                 │
│              │  제목                                           │
│              │  운동하기                                       │
│              │  (text-xl font-semibold)                        │
│              │                                                 │
│              │  메모                                           │
│              │  30분 이상 유산소                               │
│              │  (text-sm)                                      │
│              │                                                 │
│              │  등록일                                         │
│              │  2026.05.10 09:00                               │
│              │  (text-xs text-text-secondary)                  │
└──────────────┴─────────────────────────────────────────────────┘
```

**삭제 확인 모달 (role="alertdialog"):**
```
┌────────────────────────────────────────────────┐
│  할 일 삭제                               [X]  │
│  ──────────────────────────────────────────── │
│  이 작업은 되돌릴 수 없습니다.                  │
│                                                │
│  삭제를 확인하려면 ID를 입력하세요              │
│  ┌──────────────────────────────────────────┐ │
│  │                                          │ │
│  └──────────────────────────────────────────┘ │
│                                                │
│  [취소]                    [제출 (disabled)]   │
│              ← 입력값 = taskId 일 때 활성화     │
└────────────────────────────────────────────────┘
```

**404 (리소스 없음):**
```
┌──────────────┬─────────────────────────────────────────────────┐
│ LNB          │                                                 │
│              │                                                 │
│              │         리소스를 찾을 수 없습니다.              │
│              │                                                 │
│              │         [ArrowLeft  목록으로 돌아가기]          │
│              │                                                 │
└──────────────┴─────────────────────────────────────────────────┘
```
